HTML Basics

Tag Description Example
<h1> - <h6> Defines headings from most important (<h1>) to least important (<h6>). <h1>Hello World</h1>
<p> Defines a paragraph of text. <p>This is a paragraph.</p>
<a> Defines a hyperlink, used to link to other pages. <a href="https://example.com">Visit Example</a>

CSS Selectors & Properties

  • Selectors
    • element: Targets all elements of that type (e.g., p).
    • #id: Targets an element with a specific id attribute (e.g., #header).
    • .class: Targets all elements with a specific class (e.g., .button).
  • Common Properties
    • color: Sets the color of the text (e.g., color: blue;).
    • font-size: Adjusts the size of the font (e.g., font-size: 16px;).
    • margin: Adds space around an element (e.g., margin: 10px;).
